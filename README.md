# 🌸 Flower Market Website

A responsive **Flower Market** landing page built using **HTML**, **CSS**, and **Tailwind CSS (via CDN)**.  
The design showcases a flower shop with sections like Hero, Plants, Flower & Plant Lover, Deals, Subscribe, and Footer.

---

## 🔗 Live Demo

**[View Live Website](https://flowermarket.netlify.app/)**

---

## ⚙️ Features

- **Responsive Layout:** Works perfectly on desktop, tablet, and mobile.
- **Tailwind CSS:** Utility-first framework included via CDN.
- **Custom Styles:** Additional styles in `style.css`.
- **Sections Included:**
  - Header with navigation and cart.
  - Hero banner with heading and image.
  - Plants grid with "Add to Cart" buttons.
  - Flower & Plant Lover section with decorative badge.
  - Deals section with image backgrounds.
  - Newsletter subscription section.
  - Footer with navigation and social icons.

---

## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- **Tailwind CSS (via CDN)**

---

## 🚀 How to Use

1. **Download or clone this repository.**
2. Ensure all images are in the `assets/` folder.
3. Open `index.html` directly in your browser.

---

## 📱 Responsive Design

- The layout switches to a **mobile-friendly version** below 628px width.
- Plants and deals grids convert to single-column layouts.
- Navigation is hidden for smaller devices.

---

## 🎨 Customization

- **Colors:** Modify the `:root` variables in `style.css`.
- **Images:** Replace files in the `assets/` folder with your own images.
